角色特性

卡提希娅拥有独特的双形态能力——平时以温柔的治愈者形象示人，战斗时可切换为锐利的战士模式。她的星光属性赋予了她在黑暗中发光的能力，也让她成为队伍中最不可或缺的核心角色。

// Lore

故事背景

出身

拂风水畔的农家少女

卡提希娅出身于黎那汐塔拂风水畔的普通农家，原本是再平凡不过的少女。在一次「岁主感恩礼」上，她误饮供奉圣酒，酒醉后跃上祭典高台，撩起裙摆踉跄起舞，还夺过圣事号角吹响民间小调——使节们追捕不及，严肃的祭礼就此变成了狂欢闹剧。隐海修会事后将她关押了三天禁闭，她却把禁闭室的墙涂满了涂鸦。

二十年前

被设计的圣女

卡提希娅与被鸣式「利维亚坦」侵蚀异化的岁主英白拉多发生共鸣，不知情的隐海修会认定她为「圣女」，赐名「芙露德莉斯」为她加冕。然而这顶冠冕是鸣式操控她的枷锁，她的诞生与受封，不过是一场被精心设计的阴谋。第二次黑潮随之在神学院「阿维纽林」降临，据说正是她与鸣式共鸣所引发。

分裂

一分为二的意志

为了摆脱利维亚坦的控制，芙露德莉斯将自我意识一分为二：将渴望延续生命、保留本名的那部分分离为「卡提希娅」，而继承大部分力量、以毁灭一切（包括自身）来对抗鸣式的部分，则保留了「芙露德莉斯」之名。此后卡提希娅以分身形态游荡，只有漂泊者能看见她的外表与频率波动。

狂欢节

再度现世

每年狂欢节最后一日，卡提希娅会短暂降临人间。她驻足水面，模仿漂泊者在谢幕时表演的舞步。漂泊者循岁主的指引找到她，两人手掌相触的瞬间，她额心的声痕不断闪烁——暗示着两者之间存在着某种无法言说的特殊渊源。

终章

斩断枷锁

在漂泊者与坎特蕾拉的介入下，卡提希娅借助夏空的共鸣力在梦境中传递线索，引导漂泊者一步步接近真相。坎特蕾拉以神权剑「提尔芬」将力量反哺英白拉多，岁主短暂夺回控制权，让芙露德莉斯得以亲手斩下被鸣式异化的身躯。现在，她是黎那汐塔的流浪骑士，故事仍未落幕。

游戏攻略

⚔️

双形态机制

卡提希娅分「小卡」与「大卡（芙露德莉斯）」两种形态。小卡阶段通过普攻、技能分别召唤异权剑/神权剑/人权剑点亮三格回路图标，满后开共鸣解放切换大卡。大卡积攒【决意】至满触发二段大招「看潮怒风哮之刃」，是核心爆发窗口。

🌪️

风蚀效应理解

风蚀是计时型DOT，到时间整体结算，不逐层跳伤。卡提绝大多数增益直接挂钩风蚀层数，层数越高直伤与结算伤害越高。0链自挂上限5层；2链风蚀上限提升3层；3链大幅提高自挂效率；配夏空后台可稳定维持6层以上。

❤️

生命值倍率角色

鸣潮首位生命值倍率主C，技能伤害以生命值为底数计算，切勿按传统主C无脑堆攻击。毕业面板参考：生命值49000+，暴击率75%+，暴击伤害260%+，共鸣效率110%+。

🛡️

大招消耗注意

施放共鸣解放「听骑士从心祈愿」（小卡大招）会消耗50%自身生命值上限（5链后降至25%）。高难度副本务必配备回血手段，或在血量充足时再行释放，5链前打深塔存在被打死风险。

▶️

小卡输出循环

变奏入场召唤异权剑 → 从二段普攻起手×3召唤神权剑 → 放共鸣技能召唤人权剑 → 接普攻+下落攻击激活BUFF并回收三剑 → 开共鸣解放切换大卡。变奏后从二段普攻起手可节省一次攻击时间。

💥

大卡输出循环

大卡入场后标准循环：5A×2 + EE + 3A + R（二段大招）。决意满后不必急着放大，优先打完当前高伤连段再释放。大招「看潮怒风哮之刃」伤害根据目标风蚀层数追加增伤，释放前务必确保风蚀充足。

📊

技能升级权重

共鸣回路（1.0）> 常态攻击（0.65）≈ 共鸣解放（0.65）> 共鸣技能（0.12）> 变奏技能（0.05）。优先拉满共鸣回路与共鸣解放，其次常态攻击，技能和变奏小升即可。

🔄

低链/无夏空简化打法

风蚀覆盖不稳定时改用短循环：5A + EE + 3A + R，确保放大时仍有足量风蚀层数。无夏空时卡提直伤约为满配的66%；换散华约为80%。共鸣效率建议堆至110%~120%保证循环。

🗡️

武器选择

首选专武「不屈命定之冠」：满级412攻击白值+72.2%生命值，变奏/普攻后15秒无视防御，风蚀效应≥1层时伤害加深，相比次选提升约23%输出，无完全平替。次选：裁春、赫奕流明、千古洑流、不灭航路。四星武器词条全为攻击力，不适配生命C机制，均不推荐。

👻

声骸套装

首选「愿戴荣光之旅」5件套，配置44111（非传统43311）。卡提是生命C，3费固定词条是攻击力相当于亏一格，故改用两个4费补暴击/爆伤。首位声骸「共鸣回响·芙露德莉斯」，为自身和气动主提供额外气动加成；第二个4费选「梦魇·凯尔匹」。

📈

词条优先级

副词条：双暴（暴击率/暴击伤害）> 大生命% > 普攻加成 > 共鸣解放提升 > 小生命。主词条：4费选暴击率+爆伤（注意配平，暴击率不低于70%），1费全选生命加成。无专武情况下生命词条收益约提升1.4倍。

🏆

毕业面板

生命值 49000+，暴击率 75%+，暴击伤害 260%+，共鸣效率 110%~120%。技能升满优先：共鸣回路 = 共鸣解放 > 常态攻击 > 共鸣技能 ≈ 变奏技能。

👑

答案队：卡提+夏空+气动主

当前版本最强阵容，造价仅三金，性价比极高。气动主提高风蚀效应上限，夏空后台持续挂风蚀+24%气动伤害+10%减抗，让大招吃满增伤。出场顺序：气动主延奏夏空 → 夏空延奏卡提 → 卡提变奏入场。

🛡️

稳定奶盾：卡提+夏空+千咲

高难度副本推荐。千咲提供稳定奶量，卡提毕业生命约5万，配千咲后生存无忧。卡提声骸用愿戴荣光+芙露德莉斯；夏空用流云逝尽之空+梦魇·凯尔匹；千咲用命理3件+幽夜2件套。

💧

无夏空过渡：卡提+散华+气动主

散华延奏使下一位普攻伤害加深，卡提技能与解放均视为普攻伤害，有一定增益。但风蚀覆盖不稳，整体输出约为带夏空版本的80%，建议尽快补入00夏空。

🔗

链数与配队

0~2链：强绑夏空，自身挂层能力弱。3链：可脱离夏空，自由搭配其他辅助（卡提+气动主+千咲）。6链：最佳搭档仍是夏空，配合满层风蚀秒跳结算伤害极为恐怖，夏空可累计约70万风蚀结算伤害。

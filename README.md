一个专为《鸣潮》玩家打造的**角色卡提图鉴+攻略工具站**，基于前端三件套开发，轻量、高效、无冗余依赖，一站式解决角色玩法问题！

## 🎮 项目介绍
### 核心功能
- 📸 **高清角色图鉴**：全角色立绘/建模/突破立绘，支持高清下载
- 📋 **角色卡提详情**：技能/共鸣链/属性/定位/专属武器全解析
- 📝 **养成攻略**：升级材料/声骸搭配/词条优先级/养成成本计算
- 🎯 **配队推荐**：不同阵容组合思路，适配不同玩法场景
- 📱 **响应式设计**：PC/手机/平板全端适配，随时随地查攻略
- ⚡ 轻量无冗余：纯前端开发，加载速度快，无后端依赖

### 技术栈（纯前端三件套）
- 🔤 结构：HTML5（语义化标签，SEO友好）
- 🎨 样式：CSS3（Flex/Grid 布局 + 媒体查询适配 + 自定义动画）
- 🚀 交互：JavaScript（原生JS实现所有交互，无框架依赖，轻量化）
